# Provider Pitch Demo

عرض تفاعلي لتقديم منصة **هانتور** لمكاتب تأجير السيارات في سوريا.

## كيف يعمل

- شاشة تعرض صورة هانتور (شعار + اسم + شعار تعريفي) مغطّاة بشبكة مربعات.
- يُطرح سؤال واحد في كل مرة من نوع: «هل تواجه هذه المشكلة؟».
- إجابة **نعم** → يُكشف مربع عشوائي من الصورة.
- إجابة **لا** → ينتقل للسؤال التالي مع اهتزاز خفيف.
- في النهاية تظهر الصورة كاملة مع الشعار التعريفي.

## تشغيل محلي

لا يوجد build step. افتح `index.html` مباشرة في المتصفح، أو شغّل خادم بسيط:

```bash
python3 -m http.server 8080
# ثم افتح http://localhost:8080
```

> ملاحظة: فتح الملف مباشرة بـ `file://` قد يمنع تحميل `questions.json` بسبب CORS — استخدم خادم محلي.

## التخصيص

- **الأسئلة:** عدّل `questions.json`. حجم الشبكة يتكيّف تلقائياً مع عدد الأسئلة.
- **الصورة:** ضع ملف `assets/hantoor.png` (مربعة الأبعاد، شعار + اسم + شعار تعريفي).
- **الألوان:** متغيّرات CSS في أعلى `styles.css`.

## اختصارات لوحة المفاتيح

- `→` أو `Y` → نعم
- `←` أو `N` → لا
