# Assets

Place the final Hantoor image here as `hantoor.png`.

Recommendations:
- Square aspect ratio (1:1) — the stage is a square.
- Composition: logo on top, "هانتور" name in the middle, slogan below.
- High contrast against the dark cover so partial reveals are readable.
- PNG with transparent or white background works best.

Until you drop the real file in, the demo will show a broken-image placeholder.
